Программа позволяет вести статистику по коммунальным расходам, отслеживать и анализировать динамику расходов по счётчикам.

Приложение разбито на разделы, переход между которыми осуществляется через главное меню. Вызвать его можно нажатием на значок в верхнем левом углу или движением пальца от левого края экрана (на мобильных устройствах).
Учет может вестить в разрезе нескольких квартир. При первом запуске автоматически создается квартира по умолчанию, которую затем можно будет переименовать. Все данные сохраняются с привязкой к текущей квартире, которая отображается в главном меню.

Перед началом ведения учета необходимо настроить программу, внести все необходимые счетчики и услуги.


Счетчики

В разделе "Счетчики" необходимо добавить все приборы учета, по которым будет вестить учет. Для этого нужно нажать кнопку "+" на нижней панели и заполнить форму. Форма состоит из двух вкладок. На первой можно выбрать тип счетчика, указать его название (будет использоваться для отображения в списке), начальные показания (на момент внесения счетчика), признак учета общедомовых нужд. На второй - настроить тариф. Двухставочный и трехставочный тариф используются в тех случаях, когда на потребление ресурса установлена социальная норма. Позднее можно будет отредактировать все поля счетчика, кроме поля "Тип счетчика". После заполнения всех данных необходимо сохранить их с помощью кнопки с галочкой на нижней панели (в дальнейшем эта кнопка будет встречаться на всех формах ввода данных). После создания счетчиков они появятся в списке. Напротив каждого будут указаны последние внесенные показания по этому счетчику. С помощью длительного нажатия на счетчик можно вызвать контекстное меню, которое содежит два пункта: "Изменить" и "Удалить". При нажатии на счетчик происходит переход на страницу со списком всех показаний данного счетчика с момента его создания, упорядоченными по убыванию даты. Их можно редактировать и удалять с помощью кнопок нижнего меню, предварительно выделив нужные показания в списке. Добавлять показания через эту страницу не рекомендуется, этот способ использовался в первых версиях приложения и уже устарел. Учет показаний счетчиков следует вести в разделе "Счета".


Услуги

Следующий раздел - "Услуги". Сюда заносятся все услуги, для которых не установлены или не предназначены счетчики. На форме создания услуги имеются четыре способа расчета стоимости услуги: фиксированная стоимость, по количеству жильцов/площади (сюда попадают все услуги, расчет стоимости по которым ведется по формуле "тариф*коэффициент"),  водоотвод (необходимо выбрать из перечня список счетчиков,  из которых будут браться данные), подогрев воды. Изменить способ расчета в дальнейшем будет нельзя.


Счета

Когда настроен список квартир, внесены все счетчики и услуги, можно переходить к ведению учета. Осуществляется он в разделе "Счета".
Счет представляет собой набор показаний счетчиков и список услуг. При создании счета нельзя использовать уже существующие в программе показания счетчиков, нужно добавить новые. Можно пропустить часть счетчиков (не добавляя показания по ним) и часть услуг (нажать на услугу и дождаться, пока она станет неактивной). В дальнейшем можно будет добавить пропущенные показания и услуги в режиме редактирования счета (счет открывается на просмотр и редактирование при нажатии на него в списке счетов). Для этого необходимо нажать на кнопку "Показать пустые" на нижней панели и внести недостающие данные, после чего сохранить счет. При нажатии на ранее добавленные показания счетчика, они откроются для редактирования. В процессе редактирования/добавления счета можно изменять тарифы счетчиков и услуг. Если тариф изменяется в режиме добавления нового счета - он применится к текущим показаниям счетчика и сохранится на будущее. Если же тариф изменяется в режиме редактирования счета - он изменится только у данных показаний счетчика/услуги.

На новые показания счетчиков накладывается ряд ограничений. Дата новых показаний должна быть больше даты начальных показаний. Нельзя создать несколько показаний с одинаковой датой. Сами показания должны быть не меньше предыдущих, которые высчитываются по дате и отображаются в нижней части страницы. В поле "Показания" можно добавлять как абсолютное (текущие показания счетчика), так и относительное (разница между текущими показаниями и предыдущими) значение с ведущим знаком "+", например, "+3.65".


Напоминания

В разделе "Напоминания" можно создавать уведомления, которые будут срабатывать автоматически в фоновом режиме и оповещать вас с помощью указанного текста. Напоминания могут срабатывать каждый месяц в указанную дату и время, если установлен флаг "Повторять ежемесячно", или же один раз, если флаг снят.


Экспорт

В разделе "Экспорт" можно отправить данные по SMS или электронной почте, выгрузить данные в файл, совместимый с MS Excel, сделать резервную копию и восстановить данные из нее. Выгрузка файлов происходит в учетную запись OneDrive, привязанную к телефону. Этот раздел недоступен в бесплатной версии приложения.
